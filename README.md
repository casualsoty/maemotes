### Settings
#### Close Dialog Window
Configure whether or not the dialog window will be closed after sending an emote.

`default: true`
#### Enable Chat Bubbles
Configure whether or not chat bubbles will be displayed above Tokens for maemote type chat messages.

`default: true`
